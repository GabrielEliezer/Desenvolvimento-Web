Linguagem de Programação

Esse exercicio é o ultimo da primeira lista desenvolvida na atividade de Lógica de programação, nessa atividade nós programamos um código que recebe
dois valores, x1 e x2 e calculamos a sua equação do segundo grau que seriam as suas raizes.




Nesse programa montamos um algorito que ordena os valores informados, ele verifica se o valor é maior que o outro, se for ele troca os valores,
se não ele mantem o mesmo e testa outro.




Nesse exercicios, simulamos uma máquina de café, onde temos 4 opções, cada opção tem um valor, o usuario insere uma opção e ela cai dentro
de um switch case, cada um dos cases tem um retorno para o troco do cliente.




Nesse exercicios desenvolvemos uma sequencia de Fibonacci, pedimos para o usuario inserir até quanto gostaria que a sequencia fosse e montamos a lógica
para isso. Basicamente o programa é composto pelo while da imagem.




Nesse exercicios desenvolvemos uma logica utilizando for e os indices de uma matriz para descobrir a determinante de uma matriz, basicamente usamos o momento onde o indice tinha o mesmo valor de linha e coluna da matriz para imprimirmos os valores da determinante.




Montamos o algoritmo para realizar uma busca binaria, inserimos um vetor com seus valores e percorremos ele buscando o item que nós
gostariamos e também mostrei o index onde o valor estava.




Usei o que foi ensinado sobre recursividade para tentar resolver essa atividade, esse exercicio precisa inserir um valor maior do que 0 e puxamos uma função que realiza a soma de todos os valores que tem de de 1 até 5, se colocarmos 5 como entrada obteremos um valor
de 15 como resultado na saida.




Nessa atividade desenvolvemos um protetor de tela, onde utilizamos nosso conhecimento de POO para desenvolver esse protetor, utilizamos tambérm JavaFX para desenvolver as classes elipse, rectangle entre outras que foram desenvolvidas.
