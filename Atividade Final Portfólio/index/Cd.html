Nessa imagem, iniciamos o desenvolvimento de gráficos por meio de um dataframe. Nessa imagem mostro os comando que usamos para gerar um gráfico.
O comando df.groupby(column).count().plot(y='name', kind='bar', figsize=[15,5], legend=False) utiliza a biblioteca pandas em Python para agrupar um DataFrame pela coluna especificada, contar os elementos de cada grupo, e depois plotar um gráfico de barras com os resultados.
