Para desenvolver essa atividade trabalhamos bastante com Javascript, onde criamos os campos para inserir informação, atribuimos um id e utilizamos esse id no Javascript, criamos elementos TD e TR para trabalhar em formato de tabela, sempre que cadastravamos um elemento, criavamos um novo td e uma nova linha para a tabela preenchendo com o valor do formulario.

Essa atividade nós criamos um cadastro de tarefas utiizando o Javascript e um pouco de html e css. Inserimos uma tarefa e um cronometro que mostra quanto tempo ela vai durar. Também adicionamos a funcionalidade de selecionar a tarefa como Não iniciada, Em andamento e concluída. Adicionamos também um botão para excluir a tarefa, cada tarefa adicionada é colocada em baixo da outra ao lado direito da tela.




Nessa atividade nós trabalhamos com o Javascript, onde manipulamos alguns elementos do DOM (Document Object Model), o Dom é basicamente todo o arquivo index que criamos do html, quando usamos o comando documet nos referimos a um arquivo html que seria o nosso "DOM", talvez eu nao tenha explicado da melhor forma porém o DOM podemos imaginar como uma arvore onde cada elemento do html fica, e com o Javascript nós podemos manipular esse DOM, adicionando elementos, classes e o que for preciso.
